# PetPal

**Live Site:** https://petpal-vercel.vercel.app/

PetPal is a full-stack pet adoption platform that connects animal shelters with people looking to adopt. Shelters can create and manage pet listings, review applications, and communicate with potential adopters all in one place. Pet seekers can browse available animals, filter by species, breed, and age, and submit adoption applications directly through the platform.

## Features

- **For Shelters** — Create and manage pet listings, approve or decline adoption applications, message applicants, write blog posts, and manage your shelter profile
- **For Pet Seekers** — Browse and search available pets, submit adoption applications, track application status, and message shelters
- **Notifications** — Real-time notifications for application updates, new messages, and comments
- **Shelter Blogs** — Shelters can publish blog posts that seekers can read and comment on

## Tech Stack

**Backend** — Django REST Framework, PostgreSQL, JWT authentication, hosted on Render

**Frontend** — React, Material UI, React Router, hosted on Vercel
